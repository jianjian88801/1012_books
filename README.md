# 基于javeweb实现的图书借阅管理系统

#### 介绍
基于javeweb实现的图书借阅管理系统，前端采用jspl+jquery，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及企业级项目的入门。


#### 软件架构
前端：jsp | js | css | jquery  
后端：servlet | jdbc  
环境：jdk1.7+ | mysql | tocmat
